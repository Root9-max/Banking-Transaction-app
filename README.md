# Banking Transaction App

## Overview
A mobile banking application developed as part of a varsity group project.
The app allows users to authenticate, view account details, and perform basic
financial transactions.

## Tech Stack
- Java
- SQL
- Android Studio
- REST APIs

## Features
- User authentication
- Account balance viewing
- Transaction processing
- API integration
